## Rodando localmente

> Clone esse repositorio 
```
git clone git@github.com:Vitosoaresp/clone_netfix.git
```

> Entre na pasta do projeto e instale as dependências
```
cd clone_netflix/ && npm install
```

> Inicie a aplicação

```
npm run dev
```
