- Feito com React e Tailwind CSS

## Instalando Dependências

> Clone esse repositorio 
```
git clone git@github.com:Vitosoaresp/clone-netfix.git
```

> Entre na pasta do projeto e instale as dependências
```
cd clone-netflix/
npm install
```

> Inicie a aplicação

```npm run dev ```
